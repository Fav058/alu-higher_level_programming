Test cases for the MyList class
================================

Import the class
-----------------

    >>> MyList = __import__('1-my_list').MyList

Empty list
----------

    >>> my_list = MyList()
    >>> print(my_list)
    []
    >>> my_list.print_sorted()
    []

Basic usage
-----------

    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Already sorted list
--------------------

    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.print_sorted()
    [1, 2, 3]

List with duplicates
---------------------

    >>> my_list = MyList()
    >>> my_list.append(5)
    >>> my_list.append(5)
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1, 5, 5]

List with negative numbers
----------------------------

    >>> my_list = MyList()
    >>> my_list.append(-3)
    >>> my_list.append(0)
    >>> my_list.append(-10)
    >>> my_list.print_sorted()
    [-10, -3, 0]
